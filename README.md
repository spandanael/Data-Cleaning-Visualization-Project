# Data Cleaning and Visualization using Titanic Dataset

## Project Overview
This project focuses on cleaning and visualizing the Titanic dataset using Python. It demonstrates data preprocessing techniques such as handling missing values, checking duplicate records, and creating visualizations to understand the data.

## Tools and Technologies
- Python
- Google Colab
- Pandas
- Matplotlib
- Seaborn

## Project Steps
- Loaded the Titanic dataset.
- Checked dataset information.
- Handled missing values.
- Checked duplicate records.
- Created visualizations:
  - Survival Count
  - Gender Distribution
  - Age Distribution
  - Passenger Class Distribution
  - Survival by Gender
  - Correlation Heatmap

## Outcome
The dataset was successfully cleaned and visualized, providing useful insights into passenger demographics and survival patterns.
